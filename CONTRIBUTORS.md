## How to Get Listed Here

Contribute to QRMessenger in any of these ways:

1. **Submit Code** - Create a pull request with features or fixes
2. **Report Bugs** - Open an issue with detailed reproduction steps
3. **Improve Documentation** - Fix typos, clarify instructions, add examples
4. **Suggest Features** - Open a discussion with ideas
5. **Translate** - Add support for other languages
6. **Spread the Word** - Star, share, recommend

---

## Recognition Levels

### 🥇 **Major Contributors** (10+ commits)
- Core features
- Security improvements
- Significant code refactoring

### 🥈 **Regular Contributors** (5-9 commits)
- Bug fixes
- Documentation improvements
- Feature enhancements

### 🥉 **Community Contributors** (1-4 commits)
- Bug reports with fixes
- Documentation typos
- Small improvements

### ⭐ **Supporters**
- Issues that improve the project
- Design feedback
- Translation efforts

## How to Contribute

## Community Guidelines

- **Respect**: Treat everyone with kindness and respect
- **Inclusivity**: Welcome contributions from all backgrounds
- **Transparency**: Be clear about your intent and changes
- **Constructive**: Focus on ideas, not on people
- **Open-minded**: Listen to feedback and suggestions
